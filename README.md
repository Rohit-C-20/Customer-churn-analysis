Customer Churn Analysis – Telecom Industry
Project Overview

Customer churn is a major challenge for telecom companies, as losing customers directly affects revenue and growth. This project analyzes telecom customer data to identify patterns and factors that contribute to customer churn. Using Python for data analysis and Power BI for visualization, the project provides insights into customer behavior and highlights areas where businesses can improve customer retention.

Dataset

Dataset used: Telco Customer Churn Dataset (Kaggle)

The dataset contains customer demographic information, service subscriptions, and billing details. Key columns include:

CustomerID

Gender

SeniorCitizen

Contract Type

Internet Service

Monthly Charges

Total Charges

Payment Method

Churn (Yes/No)

Tools & Technologies

Python (Pandas, NumPy, Matplotlib, Seaborn) – Data cleaning and analysis

Power BI – Interactive dashboard creation

Google Colab – Running Python analysis

GitHub – Version control and project documentation

Project Workflow
1. Data Cleaning

Converted data types (e.g., TotalCharges to numeric)

Removed missing values

Checked dataset structure and summary statistics

2. Exploratory Data Analysis (EDA)

Performed analysis to understand customer churn patterns such as:

Churn distribution

Contract type vs churn

Internet service vs churn

Monthly charges vs churn

Payment method vs churn

3. Dashboard Development

A Power BI dashboard was created to visualize key business insights including:

Total Customers

Total Churned Customers

Churn Rate

Churn by Contract Type

Churn by Internet Service

Churn by Payment Method

Average Monthly Charges by Churn

Key Insights

Customers with month-to-month contracts have the highest churn rate.

Higher monthly charges increase churn probability.

Customers using electronic check payment methods churn more frequently.

Fiber optic internet users show slightly higher churn compared to DSL users.

These insights can help telecom companies develop targeted retention strategies.
